module Shrine::Plugins::DeleteUploaded

  1. lib/shrine/plugins/delete_uploaded.rb

The delete_uploaded plugin will automatically delete files after they have been uploaded. This is especially useful when doing processing, to ensure that temporary files have been deleted after upload. One exception are Shrine::UploadedFile files, they won't get deleted for stability reasons.

plugin :delete_uploaded

By default this behaviour will be applied to all storages, but you can limit this only to specified storages:

plugin :delete_uploaded, storages: [:store]
