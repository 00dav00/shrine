module Shrine::Plugins::RemoveInvalid

  1. lib/shrine/plugins/remove_invalid.rb

The remove_invalid plugin automatically deletes a cached file if it was invalid and deassigns it from the record. If there was a previous file attached, it will be assigned back, otherwise nil will be assigned.

plugin :remove_invalid
