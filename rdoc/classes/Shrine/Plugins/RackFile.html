module Shrine::Plugins::RackFile

  1. lib/shrine/plugins/rack_file.rb

The rack_file plugin enables models to accept Rack file hashes as attachments.

rack_file #=>
# {
#   filename: "cats.png",
#   type: "image/png",
#   tempfile: #<Tempfile:/var/folders/3n/3asd/-Tmp-/RackMultipart201-1476-nfw2-0>,
#   head: "Content-Disposition: form-data; ...",
# }
user.avatar = rack_file
user.avatar.original_filename #=> "cats.png"
user.avatar.mime_type         #=> "image/png"

Internally the plugin wraps the Rack file hash into an IO-like object, and this is what is passed to Shrine#upload.

plugin :rack_file

Note that this plugin is not needed in Rails applications, because Rails already wraps Rack uploaded files in ActionDispatch::Http::UploadedFile.
