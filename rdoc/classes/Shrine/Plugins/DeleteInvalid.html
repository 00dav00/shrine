module Shrine::Plugins::DeleteInvalid

  1. lib/shrine/plugins/delete_invalid.rb

The delete_invalid plugin immediately deletes the assigned attachment if it failed validation.

plugin :delete_invalid

By default an attachment is always cached before it's validated. This way the attachment will persist when the form is resubmitted, which is consistent with the other fields in the form. However, if this is a concern, you can load this plugin.
