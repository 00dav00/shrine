module Shrine::Plugins::CachedAttachmentData

  1. lib/shrine/plugins/cached_attachment_data.rb

The cached_attachment_data plugin adds the ability to retain the cached file across form redisplays, which means the file doesn't have to be reuploaded in case of validation errors.

plugin :cached_attachment_data

The plugin adds #cached_<attachment>_data to the model, which returns the cached file as JSON, and should be used to set the value of the hidden form field:

<%= form_for @user do |f| %>
  <%= f.hidden_field :avatar, value: @user.cached_avatar_data %>
  <%= f.field_field :avatar %>
<% end %>
