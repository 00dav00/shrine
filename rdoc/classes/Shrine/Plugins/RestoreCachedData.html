module Shrine::Plugins::RestoreCachedData

  1. lib/shrine/plugins/restore_cached_data.rb

The restore_cached_data plugin ensures the cached file metadata hasn't been tampered with, which can be done by modifying the hidden field before submitting the form. The cached file's metadata will be reextracted during assignment and replaced with potentially tampered one.

plugin :restore_cached_data
