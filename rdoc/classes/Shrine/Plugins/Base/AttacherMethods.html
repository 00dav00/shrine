assign (value)

Receives the attachment value from the form. If it receives a JSON string or a hash, it will assume this refrences an already cached file (e.g. when it persisted after validation errors). Otherwise it assumes that it's an IO object and caches it.

[show source] 
# File lib/shrine.rb, line 482
def assign(value)
  if value.is_a?(String)
    assign_cached(value) unless value == ""
  else
    uploaded_file = cache!(value, phase: :cache) if value
    set(uploaded_file)
  end
end

# File lib/shrine.rb, line 530
def promote(cached_file, phase: :store)
  stored_file = store!(cached_file, phase: phase)
  result = swap(stored_file) or delete!(stored_file, phase: :stored)
  result
end

  else
    default_url(**options)
  end
