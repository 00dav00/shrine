assign (value)

Checks whether a file is a Rack file hash, and in that case wraps the hash in an IO-like object.

[show source] 
# File lib/shrine/plugins/rack_file.rb, line 30
def assign(value)
  if value.is_a?(Hash) && rack_file?(value)
    assign(UploadedFile.new(value))
  else
    super
  end
end
